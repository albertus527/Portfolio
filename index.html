Ikeh


Di project ini tim saya yang berisikan 4 orang termasuk saya, membuat sebuah web yang berisikan tentang "Toko Furniture Custom"
User atau pengguna bisa memesan furnitur sesuai keinginan mereka, mulai dari memodifikasi model,warna dan bahan yang akan digunakan Di project ini saya bertugas untuk membuat bagian "Front End" atau tampilan dari web tim kami.

Project ini memiliki fitur "Login / Logout user" dan juga, project ini memiliki fitur Cart yang akan aktif ketika user sudah login
Cart ini menggunakan sistem "Local Storage" yang dimana isi cart tidak akan hilang walaupun user sudah me-refresh halaman web.
Isi Cart akan hilang ketika user sudah memilih tombol "Checkout" atau memilih tombol "Hapus Pesanan"
